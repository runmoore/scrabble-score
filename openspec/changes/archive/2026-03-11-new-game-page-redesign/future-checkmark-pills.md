## Checkmark icon on selected pills

Add a small checkmark icon to selected player and game type pills, in addition to the color change. This would provide a secondary visual indicator of selection state, improving clarity — especially for users with color vision deficiencies.

**Considerations:**

- Use an inline SVG or a Tailwind-compatible icon library
- Position the checkmark to the left of the pill text
- Animate the checkmark appearing/disappearing with a subtle transition
- Ensure the icon doesn't increase the pill's touch target size unexpectedly
